IAssembly#

class IAssembly#

Bases: ansys.modelcenter.workflow.api.irenamable_elements.IRenamableElement, ansys.engineeringworkflow.api.IControlStatement, ansys.modelcenter.workflow.api.igroup.IGroupOwner, IAssemblyChild, abc.ABC

Represents a ModelCenter assembly.

Assemblies organize components and other assemblies in the workflow. Additionally, assemblies can have datapins appended to themselves, allowing them to act as a way to abstract subordinate parts of the model.

Each ModelCenter workflow has an assembly as its root element, containing all other assemblies.
